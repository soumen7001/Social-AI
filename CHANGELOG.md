# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-05

### Added
- Initial release of SocialAI.
- AI-powered Reels analysis using Gemini 2.0.
- Local image classification with ResNet-50.
- Firebase integration for Auth and History.
- Premium Glassmorphic UI with Tailwind CSS.
- Fallback mechanisms for Poe API (Claude 3).

### Fixed
- Improved API routing for vision tasks.
- Enhanced error handling for model downloads.
- Optimized frontend performance for image previews.
